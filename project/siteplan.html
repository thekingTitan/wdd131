Site Plan: Personal Recipe Book

Site Name

Proposed Name: MyRecipeBook

The name "MyRecipeBook" was chosen because it is simple, relatable, and clearly conveys the purpose of the website: a personal repository for recipes.

Site Purpose

The purpose of this site is to provide users with a platform to store, organize, and search their personal recipes. It is designed to make it easy to add new recipes, categorize them, and quickly find desired recipes through a search feature.

Scenarios

  • How can I add a new recipe to my collection?
  • What recipes have I categorized under "Desserts"?
  • How can I search for recipes that use "chicken" as an ingredient?

Color Schema

Primary Color: Green (#4CAF50) - Used for headings and accents.

Secondary Color: Light Gray (#f9f9f9) - Used for the background to provide a clean and subtle look.

Text Color: Dark Gray (#333) - Ensures readability for body text.

Typography

Font 1: Arial (Sans-Serif) - Used for all body text and general content.

Font 2: Georgia (Serif) - Used for headings to provide a contrasting, elegant style.
